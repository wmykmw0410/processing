## PVectorクラス
### 座標や速度など、xとyの値を同時に扱う時に使用する

```python
# PVector(xの初期値, yの初期値)
p = PVector(100, 130)
print(p.x)

p.x = 200　# プロパティへの代入
print(p.x) # プロパティの値の参照

# p = PVector()でp.xとp.yを初期化される

```

### 変数の更新とプロパティの更新
#### 変数pのプロパティ(p.x)を書き換えているだけで、変数pそのものを書き換えていないため、global命令が不要
```python
p = PVector(200, 100)


def setup():
    size(400, 200)


def draw():
    background(0)
    p.x = p.x + 2
    if p.x > 400:
        p.x = 0
    fill(0,255,0)
    ellipse(p.x, p.y, 30, 30)
```
